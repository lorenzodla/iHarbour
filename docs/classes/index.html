hButton

Class for creating UIButton.

CreateButton( (String)title, (par)x, (par)y, (par)width, (par)height ) -> hButton

  • title: (String) Title of the button.
  • x: (par) x position on the view.
  • y: (par) y position on the view.
  • width: (par) Width of the button.
  • height: (par) Height of the button.

Functions

  • Button_SetTitle( (hButton)btn, (String)title ) Changes the title of the button.
  • Button_SetBackgroundColor ( (hButton)btn, (par)red, (par)green, (par)blue, (par)alpha ) Changes the background color of the button.
  • Button_SetCorners( (hButton)btn, (par)pixels ) Set the button corners rounded.

  • Label_SetText( (hLabel)lbl, (String)text ) Set the label text.
